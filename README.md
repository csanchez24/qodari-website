# Qodari Website

Marketing website for Qodari, built with Next.js and Sanity.

This project is focused on presenting Qodari’s software solutions through a flexible content structure, reusable sections, and maintainable page architecture.

## Overview

The website is designed to support marketing pages, solution pages, and structured content management through Sanity. It uses a reusable component-based approach so content can be managed consistently across different sections of the site.

## Features

- Next.js website architecture
- Sanity content management
- Reusable page sections
- Solution-focused page structure
- Flexible content modeling for marketing pages

## Why I Built It

I built this project to create a scalable marketing website that could support multiple Qodari solutions while keeping the content structure reusable, organized, and easy to maintain.

## Tech Stack

- TypeScript
- Next.js
- React
- Sanity
- Tailwind CSS

## Current Status

This project is in active development and continues to evolve as new pages, content models, and solution content are added.
